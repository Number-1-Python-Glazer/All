Welcome , here will be mainly just the documentation from different labs ive done (and just documentation in general) at the google cybersecurity course , this course entails:

COURSE 1. FOUNDATIONS OF CYBERSECURITY
A course focused on the core principles of cybersecurity. It outlines security concepts, common threats, basic risk ideas, and the general responsibilities carried by security analysts.

Modules
Introduction to cybersecurity
Security domains and frameworks
Threats, attack types, vulnerabilities
CIA triad and risk concepts
Analyst roles and responsibilities
Professional practice tasks

COURSE 2. PLAY IT SAFE, MANAGE SECURITY RISKS
A course centred on risk management. It explains assets, threats, controls, and the processes used to identify and reduce risk inside an organisation.

Modules
Assets, threats, and impact
Risk assessment steps
Security controls and mitigations
Governance, policies, and standards
NIST CSF overview
Risk reporting activities

COURSE 3. CONNECT AND PROTECT, NETWORKS AND NETWORK SECURITY
A course focused on networking and security. It explains how data moves across networks, how protocols operate, and how security tools defend networked environments.

Modules
Networking foundations
TCP IP model
Key protocols such as DNS, HTTP, SMTP
Subnets, routing, and addressing
Firewalls, proxies, and access control
Network monitoring fundamentals
Common network attack examples

COURSE 4. TOOLS OF THE TRADE, LINUX AND SQL
A course based on Linux, command line operations, file permissions, and MySQL fundamentals. It teaches how logs are stored and investigated in real environments.

Modules
Linux structure and navigation
File systems
Permissions and ownership
Log file review
Bash fundamentals
SQL basics
Filtering, sorting, and logic
SQL for investigative tasks

COURSE 5. ASSESSMENT, IDENTIFICATION, AND ANALYSIS
A course focused on reviewing alerts and logs from different security tools. It introduces triage processes, indicators, and incident classification.

Modules
Security monitoring concepts
SIEM platform basics
Event triage workflow
Indicators of compromise
Alert investigation processes
Reporting and documentation

COURSE 6. SOUND THE ALARM, DETECTION AND RESPONSE
A course based on incident response. It explains SOC workflows, escalation stages, and standard procedures for containing and resolving security incidents.

Modules
SOC team structure
Escalation paths
Incident response stages
Containment methods
Eradication and recovery procedures
Documentation requirements

COURSE 7. AUTOMATE CYBERSECURITY TASKS WITH PYTHON
A course focused on Python for security operations. It teaches basic syntax, loops, conditions, functions, and simple automation for repetitive analysis tasks.

Modules
Python fundamentals
Variables and data types
Loops and conditions
Functions
File handling
Log parsing
Report automation
Basic security scripting

COURSE 8. PUTTING IT ALL TOGETHER, FINAL PORTFOLIO PROJECT
A course structured around a complete investigation project. It uses logs, SQL, documentation, and reasoning to produce a final case study suitable for a professional portfolio.

Modules
Scenario selection
Log analysis
Evidence development
SQL query creation
Documentation preparation
Final security report
